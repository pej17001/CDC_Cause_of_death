For the data in Q2, we downloaded this dataset from 2011 to 2020 timer period (two datasets merged), and the filters are cause of death, gender, year, and state. The state is specifically New York, and cause of death is ICD-10 Codes.

Fot the data in Q4, we downloaded two datasets. First one “Final Project Q4.txt” only contains cause of death and it’s death rate, which we used this dataset to find the top 1 causes of death. Then, we download the dataset “Final Project Q4 Full.txt” using filters cause of death, gender, census region, race,etc. We used the categorical variable from the filters to build the decision tree.
